### 相邻字符不同的最长路径

[https://leetcode-cn.com/problems/longest-path-with-different-adjacent-characters/](https://leetcode-cn.com/problems/longest-path-with-different-adjacent-characters/)

### 题目描述

<font size=2> 

给你一棵 **树**（即一个连通、无向、无环图），根节点是节点 0 ，这棵树由编号从 0 到 n - 1 的 n 个节点组成。用下标从 **0** 开始、长度为 n 的数组 parent 来表示这棵树，其中 parent[i] 是节点 i 的父节点，由于节点 0 是根节点，所以 parent[0] == -1

另给你一个字符串 s ，长度也是 n ，其中 s[i] 表示分配给节点 i 的字符

请你找出路径上任意一对相邻节点都没有分配到相同字符的 **最长路径** ，并返回该路径的长度

</font>

<font size=2>

```
输出：3
解释：任意一对相邻节点字符都不同的最长路径是：0 -> 1 -> 3 。该路径的长度是 3 ，所以返回 3 。
可以证明不存在满足上述条件且比 3 更长的路径。
```

```
输入：parent = [-1,0,0,0], s = "aabc"
输出：3
解释：任意一对相邻节点字符都不同的最长路径是：2 -> 0 -> 3 。该路径的长度为 3 ，所以返回 3 。
```

<font size=2>**提示：**</font>

- `n == parent.length == s.length`
- `1 <= n <= 10^5`
- `对所有 i >= 1 ，0 <= parent[i] <= n - 1 均成立`
- `parent[0] == -1`
- `parent 表示一棵有效的树`
- `s 仅由小写英文字母组成`

### 题解

#### 方法一

- <font size=2>如果没有相邻节点不能相同的限制，那么本题就是让求二叉树的直径上节点个数</font>
- <font size=2>枚举树中所有节点 u，计算以该节点为根并且 **合法** 的最长路径的长度</font>
- <font size=2>以 u 为根的最长路径最长，意味着左子树和右子树的路径最长，然后再 u 和左右子树根节点的字符是否不同即可</font>

#### AC代码

```
class Solution {
    int res;
    public int longestPath(int[] parent, String s) {
        int n = parent.length;

        // 初始化图
        List<Integer>[] graph = new List[n];
        for (int i = 0; i < n; i ++) graph[i] = new ArrayList<>();
        for (int i = 1; i < n; i ++) graph[parent[i]].add(i);

        dfs(graph, s, 0);

        return res;
    }

    private int dfs(List<Integer>[] graph, String s, int u) {
        // max1 max2 分别代表左右子树中路径较大值和较小值
        int max1 = 0, max2 = 0;
        for (int v : graph[u]) {
            int num = dfs(graph, s, v);
            if (num > max1 && s.charAt(v) != s.charAt(u)) {
                max2 = max1;
                max1 = num;
            } else if (num > max2 && s.charAt(v) != s.charAt(u)) {
                max2 = num;
            }
        }
        res = Math.max(res, max1 + max2 + 1);
        return max1 + 1;
    }
}
```

#### 复杂度分析

- <font size=2>时间复杂度：O(n^2)</font>
- <font size=2>空间复杂度：O(n^2)</font>
