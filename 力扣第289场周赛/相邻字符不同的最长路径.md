### 相邻字符不同的最长路径

[https://leetcode-cn.com/problems/longest-path-with-different-adjacent-characters/](https://leetcode-cn.com/problems/longest-path-with-different-adjacent-characters/)

### 题目描述

<font size=2> 

给你一棵 **树**（即一个连通、无向、无环图），根节点是节点 0 ，这棵树由编号从 0 到 n - 1 的 n 个节点组成。用下标从 **0** 开始、长度为 n 的数组 parent 来表示这棵树，其中 parent[i] 是节点 i 的父节点，由于节点 0 是根节点，所以 parent[0] == -1

另给你一个字符串 s ，长度也是 n ，其中 s[i] 表示分配给节点 i 的字符

请你找出路径上任意一对相邻节点都没有分配到相同字符的 **最长路径** ，并返回该路径的长度

</font>

<font size=2>

**示例 1：**

</font>

![hh](https://github.com/kunkun166/Algorithm-records/blob/leetcodecup/%E6%8F%92%E5%9B%BE/03.png)

```
输入：parent = [-1,0,0,1,1,2], s = "abacbe"
输出：3
解释：任意一对相邻节点字符都不同的最长路径是：0 -> 1 -> 3 。该路径的长度是 3 ，所以返回 3 。
可以证明不存在满足上述条件且比 3 更长的路径。 
```

<font size=2>

**示例 2：**

</font>

![hh](https://github.com/kunkun166/Algorithm-records/blob/leetcodecup/%E6%8F%92%E5%9B%BE/04.png)

```
输入：parent = [-1,0,0,0], s = "aabc"
输出：3
解释：任意一对相邻节点字符都不同的最长路径是：2 -> 0 -> 3 。该路径的长度为 3 ，所以返回 3 。
```

<font size=2>**提示：**</font>

- `n == parent.length == s.length`
- `1 <= n <= 10^5`
- `对所有 i >= 1 ，0 <= parent[i] <= n - 1 均成立`
- `parent[0] == -1`
- `parent 表示一棵有效的树`
- `s 仅由小写英文字母组成`

### 题解

#### 方法一

- <font size=2>尾数零的个数就是路径上的数字所含质因子 2 的个数和质因子 5 的个数中的较小值，即min{num{2}，num{5}}。比如 15, 12, 4, 3, 7五个数中，包含了 1 个质因子 5，4 个质因子 2，所以应该有 1 个尾数零（15120）</font>
- <font size=2>那么路径上的数字自然是越多越好，所以路径的起点和终点都应该在边界上</font>
- <font size=2>接下来可以预处理 **每一行** 质因子 2 和质因子 5 的前缀和，然后枚举每一个点作为拐角，如下图有四种情况</font>

- <font size=2> 由上图可知，我们需要从上往下，枚举左拐/右拐（1、2图），接着需要从下往上，枚举左拐/右拐（3、4图）。代码实现上我们是先枚举 **每一列**，再枚举每一行。枚举行的时候进行竖直方向的累加，根据前缀和我们就可以求出当前路径下的尾随零是多少，从而取 max 即可求得答案</font>
