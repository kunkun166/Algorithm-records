### 吃鸡蛋

[https://www.acwing.com/problem/content/4413/](https://www.acwing.com/problem/content/4413/)

### 题目描述

<font size=2> 

小明家里有 n 个鸡蛋

每天早上，小明都要吃 1 个鸡蛋作为早餐

小明家里还有一个母鸡

母鸡会在第 m, 2m, 3m … 天的 **晚上** 下蛋，每次只下 1 个蛋

请问，连续多少天以后，小明就没有早餐鸡蛋吃了

</font>

<font size=2> 

**输入格式**

一行两个整数 n, m 

</font>

<font size=2> 

**输出格式**

一个整数，表示答案

</font>

<font size=2> 

**数据范围**

所有测试点满足 1 ≤ n ≤ 100，2 ≤ m ≤ 100

</font>

<font size=2> **输入样例1：**</font>

```
2 2
```

<font size=2> **输出样例1：**</font>

```
3
```

<font size=2> 

**样例1解释**

第 1, 2 天的早晨，小明可以吃原本就有的鸡蛋

第 3 天的早晨，小明可以吃第 2 天晚上母鸡下的鸡蛋

第 4 天的早晨，小明没有鸡蛋可以吃了

所以，连续 3 天以后，小明就没有早餐鸡蛋可以吃了

</font>


<font size=2> **输入样例2：**</font>

```
9 3
```

<font size=2> **输出样例2：**</font>

```
13
```

<font size=2> 

**样例2解释**

第 1 ∼ 9 天的早晨，小明可以吃原本就有的鸡蛋

第 10, 11, 12 天的早晨，小明可以吃第 3, 6, 9 天晚上母鸡下的蛋

第 13 天的早晨，小明可以吃第 12 天晚上母鸡下的蛋

第 14 天的早晨，小明没有鸡蛋可以吃了

所以，连续 13 天以后，小明就没有早餐鸡蛋可以吃了

</font>


### 题解

#### 方法一

- <font size=2>这是道简单的模拟题，我们用 k 来表示现在是第几天，因为是早上吃鸡蛋，晚上生鸡蛋，所以对于每一天早上，我们都先判断是否已经没鸡蛋了，没有则跳出循环，有则鸡蛋数减一。然后利用 `k % m == 0` 判断今晚能否生鸡蛋，能则鸡蛋数加一。最后进入下一天继续以上操作。</font>

#### AC代码

```
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();   // 最开始有 n 个鸡蛋
        int m = sc.nextInt();   // m, 2m, 3m, ...的晚上母鸡会下蛋
        
        int k = 1;   // 从第一天开始枚举
        while (true) {
            if (n < 1) break;
            n--;
            if (k % m == 0) n++;
            k++;
        }
        
        System.out.print(k - 1);
    }
}
```

#### 复杂度分析

- <font size=2>时间复杂度：O(n)</font>
- <font size=2>空间复杂度：O(1)</font>
