### 题目描述

<font size=2> 三个倒扣着的不透明小碗排成一排。<br>随机挑选一个小碗，将一个小球置于碗中。<br>然后进行 n 次操作，编号 1 ∼ n。<br>对于第 i 次操作：</font>

- <font size=2>如果 i mod 2 = 1，则操作内容为将位于中间的碗和位于左边的碗交换位置。</font>

- <font size=2>如果 i mod 2 = 0，则操作内容为将位于中间的碗和位于右边的碗交换位置。</font>

<font size=2> 我们不妨用 0, 1, 2 来表示左、中、右三个位置。<br>n 次操作全部完成以后，装有小球的碗位于位置 x。<br>请你计算，所有操作开始前，装有小球的碗所在的初始位置。</font>

<font size=2> **输入格式**<br>第一行，一个整数 n<br>第二行，一个整数 x</font>

<font size=2> **输出格式**<br>输出一个 0 ∼ 2 的整数，表示所有操作开始前，装有小球的碗所在的初始位置
</font>

<font size=2> **数据范围**<br>前 6 个测试点满足 1 ≤ n ≤ 5<br>所有测试点满足 1 ≤ n ≤ 2×10^9，0 ≤ x ≤ 2
</font>

<font size=2> **输入样例1：**</font>

```
4
2
```

<font size=2> **输出样例1：**</font>

```
1
```

<font size=2> **输入样例2：**</font>

```
1 1
```

<font size=2> **输出样例2：**</font>

```
0
```
