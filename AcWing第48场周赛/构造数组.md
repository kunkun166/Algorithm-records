### 题目描述

<font size=2> 给定一个长度为 n 的整数数组 a[1], a[2], … , a[n]<br>请你构造长度为 n 的整数数组 b[1], b[2], … , b[n]，要求数组 b 满足：</font>

  1. <font size=2>b[1] = 0</font>
  2. <font size=2>对于任意一对索引 i 和 j（1 ≤ i, j ≤ n），如果 a[i] = a[j] 则 b[i] = b[j]（注意，如果 a[i] ≠ a[j]，则 b[i] 和 b[j] 相等与否随意）</font>
  3. <font size=2>对于任意索引 i（i ∈ [1, n − 1]），要么满足 b[i] = b[i + 1]，要么满足 b[i] + 1 = b[i + 1]</font>

<font size=2> 请计算，一共可以构造出多少个不同的满足条件的数组 b<br>由于答案可能很大，你只需要输出对 998244353 取模后的结果<br>例如，如果 a = [1, 2, 1, 2, 3]，则一共有 2 个满足条件的数组 b，分别是 b = [0, 0, 0, 0, 0] 和 b = [0, 0, 0, 0, 1]</font>

<font size=2> **输入格式**<br>第一行包含一个整数 n<br>第二行包含 n 个整数 a[1], a[2], … , a[n]</font>

<font size=2> **输出格式**<br>一个整数，表示对998244353 取模后的结果
</font>

<font size=2> **数据范围**<br>前 3 个测试点满足 2 ≤ n ≤ 5<br>所有测试点满足 2 ≤ n ≤ 2×10^5，1 ≤ a[i] ≤ 10^9
</font>

<font size=2> **输入样例1：**</font>

```
5
1 2 1 2 3
```

<font size=2> **输出样例1：**</font>

```
2
```

<font size=2> **输入样例2：**</font>

```
2
100 1
```

<font size=2> **输出样例2：**</font>

```
2
```

<font size=2> **输入样例3：**</font>

```
4
1 3 3 7
```

<font size=2> **输出样例2：**</font>

```
4
```
