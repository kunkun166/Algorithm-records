### 题目描述

<font size=2> 

给定一个长度为 n 的整数数组 a[1], a[2], … , a[n]

请你构造长度为 n 的整数数组 b[1], b[2], … , b[n]，要求数组 b 满足：

</font>

  1. <font size=2>b[1] = 0</font>
  2. <font size=2>对于任意一对索引 i 和 j（1 ≤ i, j ≤ n），如果 a[i] = a[j] 则 b[i] = b[j]（注意，如果 a[i] ≠ a[j]，则 b[i] 和 b[j] 相等与否随意）</font>
  3. <font size=2>对于任意索引 i（i ∈ [1, n − 1]），要么满足 b[i] = b[i + 1]，要么满足 b[i] + 1 = b[i + 1]</font>

<font size=2> 

请计算，一共可以构造出多少个不同的满足条件的数组 b

由于答案可能很大，你只需要输出对 998244353 取模后的结果

例如，如果 a = [1, 2, 1, 2, 3]，则一共有 2 个满足条件的数组 b，分别是 b = [0, 0, 0, 0, 0] 和 b = [0, 0, 0, 0, 1]

</font>

<font size=2> 

**输入格式**

第一行包含一个整数 n

第二行包含 n 个整数 a[1], a[2], … , a[n]

</font>

<font size=2> 

**输出格式**

一个整数，表示对998244353 取模后的结果

</font>

<font size=2> 

**数据范围**

前 3 个测试点满足 2 ≤ n ≤ 5

所有测试点满足 2 ≤ n ≤ 2×10^5，1 ≤ a[i] ≤ 10^9

</font>

```
5
```

<font size=2> **输出样例1：**</font>

```
2
```

```
2
```

### 题解

#### 方法一、并查集

- <font size=2>首先通过约束条件可以发现，如果 a[i] = a[j], 那么 b[i] ~ b[j] 之间的所有元素都相同，该区间的值有两种选择，一种是和前一个区间的值相同，另一种是比前一个区间的值多一</font>

- <font size=2>所以题目中求 b 数组的所有方案数就转化为求区间数。代码中采用哈希表来记录每一种数字最左端和最右端的位置，不过还是要做一遍区间合并以获得最终的区间数量。</font>

- <font size=2>区间合并可以用基础算法里的贪心做法，这里提供另外一种思路是并查集。假设现在有区间 [i, j]，那么把该区间的最右端作为该区间所有元素的祖宗节点。并且是用当前元素的索引与祖宗节点索引的大小关系来判断是否在并查集中的。</font>

#### AC代码

```
public class Main {
    static int mod = 998244353;
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        
        int[] a = new int[n];
        for (int i = 0; i < n; i++) a[i] = sc.nextInt();
        
        // 创建两个哈希表
        HashMap<Integer, Integer> left = new HashMap<>(); // left代表该数出现在最左端的位置
        HashMap<Integer, Integer> right = new HashMap<>();  // right代表该数出现在最右端的位置
        
        for (int i = 0; i < n; i ++) {
            right.put(a[i], i);   // right需要不断更新
            if (!left.containsKey(a[i])) {  // left只记录该数第一次出现的位置就好
                left.put(a[i], i);
            }
        }
        
        int max = 0, res = 1;
        for (int i = 0; i < n; i ++) {
            if (i > max) {
                res = res * 2 % mod;
            }
            max = Math.max(max, right.get(a[i]));
        }
        
        System.out.print(res);
    }
}
```

#### 复杂度分析

- <font size=2>时间复杂度：O(n)</font>
- <font size=2>空间复杂度：O(n)</font>
