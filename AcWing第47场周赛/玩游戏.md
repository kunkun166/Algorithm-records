### 玩游戏

[https://www.acwing.com/problem/content/4403/](https://www.acwing.com/problem/content/4403/)

### 题目描述

<font size=2> n 个小朋友围成一圈，玩数数游戏<br>小朋友们按顺时针顺序，依次编号为 1 ∼ n <br>初始时，1 号小朋友被指定为领头人<br>游戏一共会行进 k 轮<br>在第 i 轮中，领头人会从他的顺时针方向的下一个人开始，按顺时针顺序数 nums[i] 个人<br>其中，最后一个被领头人数到的人被淘汰出局，这也意味着该轮游戏结束<br>出局者的顺时针方向的下一个人被指定为新领头人，引领新一轮游戏<br>例如，假设当游戏即将开始第 i 轮时，还剩下 5 个小朋友，编号按顺时针顺序依次为 8, 10, 13, 14, 16，并且当前领头人为 13 号小朋友，nums[i] = 12，则第 i 轮游戏结束后，最后一个被数到的小朋友为 16 号小朋友，他将被淘汰出局，并且处于其下一位的第 8 号小朋友将被指定为新领头人<br>现在，请你求出每一轮次被淘汰的小朋友的编号</font>

<font size=2> **输入格式**<br>第一行包含两个整数 n, k<br>第二行包含 k 个整数 nums[1], nums[2] , … , nums[k]</font>

<font size=2> **输出格式**<br>一行，k 个整数，其中第 i 个整数表示在第 i 轮中被淘汰的小朋友的编号</font>

<font size=2> **数据范围**<br>前三个测试点满足 2 ≤ n ≤ 10<br>所有测试点满足 2 ≤ n ≤ 100，1 ≤ k ≤ n−1，1 ≤ nums[i] ≤ 10^9
</font>

<font size=2> **输入样例1：**</font>

```
7 5
10 4 11 4 1
```

<font size=2> **输出样例1：**</font>

```
4 2 5 6 1
```

<font size=2> **输入样例2：**</font>

```
3 2
2 5
```

<font size=2> **输出样例2：**</font>

```
3 2
```

### 题解

#### 方法一

- <font size=2>这道题考察的是约瑟夫环问题，可以用数组、链表、队列来模拟出来即可得到答案</font>

- <font size=2>因为数组和链表模拟都非常不方便，这里用队列来模拟(手写的队列，练习一下数据结构)</font>

- <font size=2>将 n 位小朋友按编号 1 ~ n 依次加入到队列中，假设现在是第 i 轮游戏，用 nums[i] 对此时的 tt - hh + 1 取余，求得应该先略过 count 个童鞋，然后依次把这 count 个童鞋从队头取出加入到队尾，此时队头元素就是要淘汰的那个童鞋</font>

#### AC代码

```
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt(); 
        int k = sc.nextInt();
        
        int[] nums = new int[k];
        for(int i = 0; i < k; i++) {
            nums[i] = sc.nextInt();
        }
        
        // 约瑟夫环问题，用队列来模拟
        int hh = 0, tt = -1;
        int[] queue = new int[10010];
        for(int i = 1; i <= n; i++) {
            queue[++tt] = i; 
        }
        
        for(int i = 0; i < k; i++) {
            int count = nums[i] % (tt - hh + 1);
            while(count-- > 0) {
                int temp = queue[hh++]; // 把队头元素取出来放到队尾去
                queue[++tt] = temp;
            }
            System.out.print(queue[hh++] + " ");
        }
    }
}
```

#### 复杂度分析

- <font size=2>时间复杂度：O(n^2)，除去读取、初始化操作外，就是 k 轮的游戏，最坏情况就是玩到只剩一个童鞋而且每次略过的人数最大，就是 (n - 1) + (n - 2) + ... + 2，所以最坏情况是O(n^2)</font>
- <font size=2>空间复杂度：O(n^2)</font>
