<font size=2> **输入格式**

第一行包含整数 n

第二行包含 n 个整数，表示整个数列</font>

<font size=2> **输出格式**

共一行，包含 n 个整数，其中的第 i 个数表示数列中的第 i 个数的二进制表示中 1 的个数</font>

<font size=2> **数据范围**

1 ≤ n ≤ 100000

0 ≤ 数列中元素的值 ≤ 10^9
</font>

